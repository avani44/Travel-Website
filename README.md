# Travel Website 🌍✈️

A responsive travel website built using **HTML, CSS, and JavaScript** that showcases popular travel destinations with a modern and user-friendly interface.

## 🚀 Features
- Responsive design for mobile and desktop
- Attractive travel destination sections
- Image gallery for locations
- Smooth navigation across the website
- Clean and modern UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
travel-website
│
├── index.html
├── style.css
├── script.js
├── images


## ▶️ How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in your browser
